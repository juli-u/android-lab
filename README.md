Android Concurrency Lab
Implementación de concurrencia nativa en Android mediante la gestión de Threads y Handlers. Este proyecto demuestra el manejo de tareas asíncronas para mantener la fluidez de la interfaz de usuario.
Fundamentos de Concurrencia: Uso de hilos nativos para comprender la comunicación base entre procesos.
Arquitectura: Separación de lógica de cómputo y actualización de vistas.
